# Student Management

The Student Management module oversees student profiles, groups, and assignments.

## Core Capabilities
- **Student Enrollment**: Profiles for each student linked to their library membership.
- **Student Groups**: Organized by program and year (e.g., `calendar_student_group`).
- **Student Assignments**: Managing and tracking academic assignments (`student_assignment`).

## Library Integration
- **Personalized Recommendations**: Uses the student's program and group data to suggest relevant books.
- **Reading Patterns**: Tracks borrowing trends within student groups to identify popular materials.

This module ensures that student data is central to the academic and library experience.
