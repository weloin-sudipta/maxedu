# Routine Generator

The Routine Generator module automatically creates class schedules and ensures academic routines are well-organized.

## Routine Generation
- **Class Schedules**: Detailed routines (e.g., `class_schedule`) for each student group and teacher.
- **Calendar Events**: Scheduling academic and library-related events (e.g., `calendar_event`).
- **Conflict Management**: Ensuring that no two classes or activities overlap for the same group or teacher.

## Library Support
- **Scheduled Library Hours**: Integrated routine management for classes to visit the library.
- **Automated Reminders**: Syncing routines with book due dates for automated reminders during library sessions.

This module provides the framework for time management across the institution.
