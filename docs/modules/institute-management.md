# Institute Management

The Institute Management module handles the administrative configuration of libraries within an educational institution.

## Management Scope
- **Institute Profiles**: Store and manage information about the institution.
- **Global Settings**: Configure default borrowing periods, fine rates, and renewal limits for the library.
- **Multi-tenant Handling**: Ensure that each institute operates as an independent tenant within the system.

## Key Actions
- **Institute Onboarding**: Create new institutes in the system.
- **Library Configuration**: Fine-tune specific library policies.
- **Subscription Management**: Monitor the SaaS subscription status of the institute.

This module provides the foundation for the entire library ecosystem.
