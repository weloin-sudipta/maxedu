# Library System

The Library System module is the core of Maxedu, providing a bridge between students, academic resources, and staff operations.

## Features
- **Book Inventory**: Management of library catalog (`library_book`) and physical copies (`library_book_inventory`).
- **Issuance & Returns**: Comprehensive workflow for borrowing and returning books, including automated status updates.
- **Renewal Management**: Smart book renewals with reservation checks and renewal caps.
- **Fine Management**: Automatic calculation and tracking of overdue fines.

## Workflows
1. **Student Request**: Student finds a book in the catalog and submits a request.
2. **Staff Approval**: Staff reviews and approves the request.
3. **Issuance**: Staff assigns a specific book copy to the member and sets a due date.
4. **Return/Renewal**: Student returns the book or requests a renewal if eligible.

This module is designed for accuracy, transparency, and ease of use in a busy academic environment.
