# Course Management

The Course Management module handles the definition and organization of academic programs and courses.

## Course Structure
- **Academic Programs**: Defining programs (e.g., `calendar_program`).
- **Course Catalog**: A detailed list of subjects within each program.
- **Enrolled Courses**: Tracks which students are enrolled in which courses.

## Library Connection
- **Subject-Specific Reading**: Links books to courses for easier discovery by students and teachers.
- **Course Materials**: Managing study materials (`study_material`) for each course.
- **Recommendation Scoring**: Multi-factor scoring (+4 points for course match) leverages course data.

Course Management ensures that both students and teachers have quick access to subject-relevant library resources.
