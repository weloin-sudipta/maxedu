# Target Users

Maxedu caters to three primary user groups within an educational institution:

## 1. Students
- **Primary Need**: Browse the library catalog, request books, track their borrow status, and receive personalized reading suggestions.
- **Key Features**: /library (catalog, requests, recommendations, issued books).

## 2. Teachers
- **Primary Need**: Access academic resources, track subject-specific reading materials, and suggest resources for their courses.
- **Key Features**: Teaching materials, Subject-based tracking, Recommendations.

## 3. Library Staff
- **Primary Need**: Manage book issuances, handle returns, monitor inventory, and oversee library operations.
- **Key Features**: /library/staff-issuance, /library/statistics, Book Issue/Request DocTypes.
