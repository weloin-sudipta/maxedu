# Core Features

Maxedu's core features form the backbone of the Library Management System.

## Catalog Management
- **Search & Browse**: Students can search the entire library catalog by title, author, category, or ISBN.
- **Availability Tracking**: Real-time status of books (available, issued, reserved).
- **Physical Copy Management**: Tracking unique physical copies of books with shelf location.

## Request & Issuance
- **Request Workflow**: Simple, one-click book request from the catalog.
- **Staff Approval**: Streamlined request approval for library staff.
- **Copy Assignment**: Assign specific book copies to borrowers for precise inventory tracking.
- **Due Date Automation**: Automatic due date calculation based on configurable borrow periods.

## Borrowing History
- **Issued Books List**: Students can view all their currently borrowed books and their due dates.
- **Lifetime History**: Complete history of every book a student has ever borrowed.

These core features ensure that basic library operations are fast, reliable, and user-friendly.
